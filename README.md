# Add bootstrap into angular project 
``` 
npm install bootstrap
```
add bootstrap file into .angular-cli.json file
```javascript
"styles": [
  "styles.css",
  "../node_modules/bootstrap/dist/css/bootstrap.min.css"
],
```
